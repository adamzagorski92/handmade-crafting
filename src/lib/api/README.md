# How to get data for each crypto?

1. Preparing id's for interesting crypto-currency (you can check avalible coins on `/coins/list` using curl or postman)
2. Get data from `/coins/{id}` (this endpoint refers to `/coins/list`) look at data: [idCoinsList.json](../../../data/idCoinsList.json)
3. You can get specific data for each coins: Read documentation: [The best practice](https://docs.coingecko.com/docs/common-use-cases)
